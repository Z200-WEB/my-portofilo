# ZAWE ZAW HTET - Portfolio | ポートフォリオ

A modern, interactive portfolio website built with React, showcasing cloud technologies, AI integration skills, and business system development experience.

React で構築されたモダンでインタラクティブなポートフォリオサイト。クラウド技術、AI活用スキル、業務システム開発の経験を紹介しています。

## Features | 機能

### English
- **Bento Grid Layout** - Modern glassmorphism design with dark mode
- **Bilingual Support** - Seamless EN/JP language toggle with animations
- **3D Interactive Cards** - CSS 3D transforms for photo cards
- **3D Tech Visualization** - Orbital system showing technology ecosystem
- **Fully Responsive** - Optimized for all screen sizes
- **Performance Optimized** - Lazy loading for Three.js components

### 日本語
- **Bento グリッドレイアウト** - ダークモードとグラスモーフィズムを採用したモダンデザイン
- **バイリンガル対応** - アニメーション付きのEN/JP言語切替
- **3Dインタラクティブカード** - CSS 3D変換による写真カード
- **3D技術ビジュアライゼーション** - 技術エコシステムを表現する軌道システム
- **完全レスポンシブ** - 全画面サイズに最適化
- **パフォーマンス最適化** - Three.jsコンポーネントの遅延読み込み

## Getting Started | はじめに

### Prerequisites | 前提条件

- Node.js 18+
- npm or yarn

### Installation | インストール

```bash
# Clone the repository | リポジトリをクローン
git clone https://github.com/Z200-WEB/my-protofilo.git

# Navigate to project directory | プロジェクトディレクトリに移動
cd my-protofilo

# Install dependencies | 依存関係をインストール
npm install

# Start development server | 開発サーバーを起動
npm run dev
```

### Build for Production | 本番用ビルド

```bash
npm run build
npm run preview
```

## Projects | プロジェクト

### CRM Dashboard | CRMダッシュボード
- **Problem:** Customer data scattered across spreadsheets
- **課題:** 顧客データがスプレッドシートに分散
- **Solution:** Salesforce-inspired centralized dashboard
- **解決策:** Salesforce風の一元管理ダッシュボード
- **Tech:** JavaScript, CSS, HTML

### Restaurant Ordering System | レストラン注文システム
- **Problem:** Manual order-taking causing delays
- **課題:** 手動の注文受付による遅延
- **Solution:** Table-based digital ordering system
- **解決策:** テーブル注文システム
- **Tech:** PHP, CSS, JavaScript

### Support Case Manager | サポートケース管理
- **Problem:** Customer inquiries scattered
- **課題:** 問い合わせ管理が分散
- **Solution:** Centralized dashboard for case management
- **解決策:** 対応状況を一元管理するダッシュボード
- **Tech:** Tailwind, React, JavaScript
